<SpeedRunTable game="eldenring" category="Any%" showCategorySelector={true} />

## How it works

The component accepts three parameters:

- `game`: The game identifier (automatically converted to the appropriate Speedrun.com ID)
- `category` (optional): A specific category - can be either the category ID or category name
- `showCategorySelector` (optional): Controls visibility of the category dropdown (default: `false`)

### Usage Examples

```astro
<!-- Default - no category selector, no category specified -->
<SpeedRunTable game="eldenring" />

<!-- With category selector enabled -->
<SpeedRunTable game="eldenring" showCategorySelector={true} />

<!-- With specific category and selector enabled -->
<SpeedRunTable game="eldenring" category="Any%" showCategorySelector={true} />

<!-- By category ID with selector -->
<SpeedRunTable game="eldenring" category="02qr00pk" showCategorySelector={true} />

<!-- By category name with selector -->
<SpeedRunTable game="eldenring" category="All Remembrances" showCategorySelector={true} />

<!-- Bloodborne examples -->
<SpeedRunTable game="bloodborne" showCategorySelector={true} />
<SpeedRunTable game="bloodborne" category="All Bosses" showCategorySelector={true} />

<!-- Dark Souls examples -->
<SpeedRunTable game="darksouls" showCategorySelector={true} />
<SpeedRunTable game="darksouls" category="Soul Level 1" showCategorySelector={true} />
```

### Table Structure

The component displays speedrun data in a table with the following columns:
- **Position**: Ranking (1st, 2nd, 3rd with gold/silver/bronze styling)
- **Player**: Player name
- **Date**: Run completion date
- **Platform**: Gaming platform used
- **IGT**: In-Game Time

The component fetches the appropriate speedrun data based on the game ID and category, displaying it in a formatted table. If no category is specified, the component will show data for all categories. The category selector is disabled by default and must be explicitly enabled with `showCategorySelector={true}`.

### Category Selection

When using the category selector, you can specify categories by either:
1. **Category ID**: The unique identifier from speedrun.com (e.g., `"02qr00pk"`)
2. **Category Name**: The display name of the category (e.g., `"Any%"`)

Both methods work interchangeably, making it easy to reference categories in your documentation or code.
