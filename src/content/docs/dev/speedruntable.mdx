## Elden Ring Speedruns (Default - No Category Selector)

<SpeedRunTable game="eldenring" />

## With Category Selector Enabled

<SpeedRunTable game="eldenring" showCategorySelector={true} />

## With Specific Category

<SpeedRunTable game="eldenring" category="Any%" showCategorySelector={true} />

## How it works

The component accepts three parameters:

- `game`: The game identifier (automatically converted to the appropriate Speedrun.com ID)
- `category` (optional): A specific category - can be either the category ID or category name
- `showCategorySelector` (optional): Controls visibility of the category dropdown (default: `false`)

### Table Structure

The component displays speedrun data in a table with the following columns:
- **Position**: Ranking (1st, 2nd, 3rd with gold/silver/bronze styling)
- **Player**: Player name
- **Date**: Run completion date
- **Platform**: Gaming platform used
- **IGT**: In-Game Time

The component fetches the appropriate speedrun data based on the game ID and category, displaying it in a formatted table. If no category is specified, the component will show data for all categories. The category selector is disabled by default and must be explicitly enabled with `showCategorySelector={true}`.
