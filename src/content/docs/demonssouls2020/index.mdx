**Demon's Souls** is a remake of the original [Demon's Souls](/demonssouls) by Bluepoint Games and was released in 2020. The game is only available on Playstation 5. There are no plans to release DLC yet. Pre-Order bonuses are/were available, but using these items is banned for all speedrun categories.

## Categories

- [Any%](/demonssouls2020/categories/any-percent) - Triggering the end credits as quickly as possible.

- [Any% (Current Patch)](/demonssouls2020/categories/any-percent-current-patch) - Triggering the end credits as quickly as possible on the current patch.

- [All Bosses](/demonssouls2020/categories/all-bosses) - Triggering the end credits as quickly as possible after defeating every boss enemy except the chalice dungeons in the game.

- [Miscellaneous Categories](/demonssouls2020/categories/miscellaneous-categories) - An overview of all other miscellaneous categories in Demon's Souls (2020).
